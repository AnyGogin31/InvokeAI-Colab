# Custom models
InvokeAI has its own program to install models inside the user interface. Please use this tab. I also recommend that you use a [configuration](https://github.com/AnyGogin31/InvokeAI-Colab/blob/main/INITIAL_MODELS.yaml) that is already ready and that you move it to the: `/content/db/configs` directory. 

![image](https://github.com/camenduru/InvokeAI-colab/assets/54370274/0aeecc0e-7442-478b-ab5e-b044861d5603)
You can also change the configuration yourself, so after the installation step, go to `/content/InvokeAI/invokeai/configs/INITIAL_MODELS.yaml` and edit _recommended: False to True or vice versa._
